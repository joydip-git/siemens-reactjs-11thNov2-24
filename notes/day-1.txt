TSC configuration file:
how to create?
	a. manual
	b. using tsc tool (>.\node_modules\.bin\tsc --init)

1. >npm init --yes => create package.json file (at the root of the directory)
2. >npm i typescript -D => install TSC (locally)
3. >.\node_modules\.bin\tsc --init => creates tsconfig.json file (at the root of the directory)
4. configure stc in tsconfig.json file
