###
Get all records:
method: GET
url: http://localhost:3000/products

Get single record:
method: GET
url: http://localhost:3000/products/1

add single record:
method: POST
url: http://localhost:3000/products
request body data: a product object

update existing single record:
method: PUT
url: http://localhost:3000/products/1
request body data: a product object

delete existing single record:
method: DELETE
url: http://localhost:3000/products/1
